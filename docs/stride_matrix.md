# Matriks Model Ancaman STRIDE

Pemetaan ancaman terhadap berbagai titik masuk (entry points) pada aplikasi OJS.

| Kategori Ancaman | Contoh pada OJS | Entry Point Terkait |
| :--- | :--- | :--- |
| **S**poofing | Penyerang login menggunakan akun curian atau sesi ilegal. | A1, A2 |
| **T**ampering | Mengubah metadata artikel atau konten jurnal secara ilegal. | D2 |
| **R**epudiation | Penyerang menghapus log aktivitas untuk menutupi jejak. | E3 |
| **I**nformation Disclosure | Bocornya jalur file atau versi server melalui pesan error. | Semua |
| **D**enial of Service | Menghabiskan penyimpanan server melalui upload massal. | B1-B4 |
| **E**levation of Privilege | Akun Author memanipulasi API untuk fungsi Manager. | D1, E4 |
